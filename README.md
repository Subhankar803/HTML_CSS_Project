# 🛒 Amazon Homepage Clone — HTML & CSS Project

A pixel-faithful front-end clone of the Amazon homepage, built from scratch using only **HTML5** and **CSS3** — no frameworks, no JavaScript. Created as the capstone project of the *Apna College* HTML & CSS course.

---

## 📸 Preview

> Open `HTML_CSS_Project.html` directly in your browser to see the live page.

---

## ✨ Features

- **Sticky Navigation Bar** — Amazon-style top nav with logo, delivery address, search bar, account links, and cart icon
- **Search Bar** — Category dropdown + input field + search button, styled to match Amazon's UI
- **Panel Bar** — Secondary nav row with "All", Today's Deals, Customer Service, Registry, Gift Cards, Sell, and a deals banner
- **Hero Section** — Full-width banner with a redirect message linking to Amazon India
- **Product Category Grid** — 8 responsive product cards covering:
  - Health & Personal Care
  - Interior & Furniture
  - Fashion & Clothes
  - Gaming Zone
  - Gym & Fitness
  - Electronic Appliances
  - Cosmetic Products
  - Kitchen Utensils
- **Footer** — Multi-column footer with "Back to Top", 4 link columns, logo, and copyright bar
- **Font Awesome Icons** — Used for location pin, search, and cart icons

## 🚀 Getting Started

No build tools or dependencies needed. Just clone and open.

```bash
# 1. Clone the repository
git clone https://github.com/Subhankar803/HTML_CSS_Project.git

# 2. Navigate into the project folder
cd HTML_CSS_Project

# 3. Open the file in your browser
open HTML_CSS_Project.html       # macOS
start HTML_CSS_Project.html      # Windows
xdg-open HTML_CSS_Project.html  # Linux
```
---

Or simply double-click HTML_CSS_Project.html in your file explorer.
